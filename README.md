# DEXLab Marketing Reporter

**Marketing analytics and reporting for DEXLab.**

---

## Getting Started

### Prerequisites

- Node.js 18+

### Setup

```bash
git clone https://github.com/jheller1212/DEXLabMarketingReporter.git
cd DEXLabMarketingReporter
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
